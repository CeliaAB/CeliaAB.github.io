display: inlineVersus display: inline-Block

What is display: inline?

display: inline is a declaration in css which allows different display properties to be assigned to content on the same line. For example, you may want to bold or emphasize certain words, give it a different appearance from the content before and after it. Using display: inline allows you to make it bold, apply a different colour, change the font, and so on.

What is display: inline-block?

display: inline-block is a declaration in css which allows different display properties to be assigned to content "inline" INCLUDING SETTING THE MARGIN AND PADDING. For example, you may want to not only bold or emphasize certain words, but you want the space it takes up on a page to be different from the content before or after it. Using display: inline-block gives you the additional choice of how much space it takes up vertically on the page. By using this setting, the entire line will appear with the height that has been set for that element as it's settings are accommdated. The rest of the line will be centred vertically within the height and padding settings, though not the margin settings.

Also, display: inline-block is used to show list items side by side, horizontally.

Why not use display: block?

display:block would not allow the item to which it is applied to be inline with the content before and after it. It will add a line between items so they will sit vertically and can't sit side by side, horizontally, and is one of the reasons that display: block is not used for lists that need to appear to be on the same line.
